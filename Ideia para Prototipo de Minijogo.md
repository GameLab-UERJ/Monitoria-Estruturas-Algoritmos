# Gameplay Loop

## Loop Principal

```text
1. Jogador realiza ação principal
        ↓
2. Recebe recompensa
        ↓
3. Usa recompensa para melhorar desempenho
        ↓
4. Enfrenta desafio maior
        ↓
5. Repete o ciclo
```

---

### Passo 1 — Ação Principal

### Passo 3 — Melhoria
